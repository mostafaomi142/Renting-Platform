Prerequisites
Required
  Node.js
  Express.js
  mysql2 (MySQL client for Node.js)
  MySQL Server: You must have a running MySQL server instance and a valid connection to it.

Optional
  nodemon (for auto-restarting during development)
  dotenv (for managing environment variables)

Installation:
   1. winget install OpenJS.NodeJS
   2. npm install express mysql2 nodemon dotenv

  3. Install MySQL Server
      Download and install MySQL Server using your preferred method (Installer, MySQL Workbench, XAMPP, etc.).

For connecting the database with your instance, replace the data inside util/database.js with you personal data.
